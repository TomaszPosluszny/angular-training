## Instalation

1. Clone the repository.
   git clone https://github.com/TomaszPosluszny/airline-ticket-booking-Webpack

2. Navigate to the project directory.
 cd repository

3. Install the dependencies.
  npm install

4. Start the application.
 ng serve
